# NASCAR Racing 2003 Points Site

A comprehensive points tracking website for NASCAR Racing 2003, built using Handlebars, React, ExpressJS, and MongoDB.

### Overview
This is a deployed site which allows users to upload race data from Nascar Racing 2003 Season to get championship results.

### Features

* Championship standings table with ample amount of performance data
* Pages on individual races and individual drivers performance
* Graphs that display changes in points over a season for top drivers and individual drivers

### Technologies Used

* Front-end: Handlebars, React components when needed
* Back end: ExpressJS
* Database: MongoDB

### How to Run the Project
1. Clone this repository using `git clone https://github.com/AsMurphy-student/430-project-2.git`
2. Install dependencies by running `npm install`
3. Start the server by running `npm run webpackWatch` in one shell and `npm run nodemon` in another shell
4. Access the website at [http://localhost:3000](http://localhost:3000)
